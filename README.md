# Task 2: Stock Trading Platform

## 📋 Project Description
A simulated Stock Trading Platform built in Java. Users can view live market data, buy and sell stocks, track their portfolio, and view transaction history.

## ✨ Features
- Real-time market data simulation (5 popular stocks)
- Buy and Sell stocks
- Portfolio management with current value
- Transaction history with timestamps
- File handling (Save & Load portfolio)
- Strong Object-Oriented Programming (OOP) design

## 🛠 Technologies Used
- Java (JDK 21)
- ArrayList
- File I/O (portfolio.txt)
- OOP Concepts (Classes & Objects)

## 🚀 How to Run
1. Open project in VS Code
2. Compile: `javac StockTradingPlatform.java`
3. Run: `java StockTradingPlatform`

## 📌 Key Functionalities
- View Market Data
- Buy/Sell Stocks
- Portfolio Performance
- Transaction History
